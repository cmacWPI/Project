Anime Infomation

What is anime?

Anime is a Japanese "cartoons" but in different level than American's cartoons.

Where it came from?

Anime originated from Japan.

What is first anime's name and when was it created?

The first anime name is Otogi Manga Calendar (おとぎマンガカレンダー) and is created at May 1, 1961
